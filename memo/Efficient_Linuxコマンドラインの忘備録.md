忘備録として、ここに記載していく。

## 条件付きリスト

`コマンド&&コマンド`で最初のコマンドが成功した場合のみ次のコマンドも実行。

`コマンド||コマンド`で最初のコマンドが失敗した場合に次のコマンドも実行。

```bash
cd dir && touch new.txt

cd dir || exit 1
```

## 無条件リスト

`コマンド;コマンド`とするとコマンドが順番に実行される。成功、失敗は関係なく実行される。

```bash
# 300秒後にHello World!が出力される
sleep 300; echo 'Hello World!'
```
